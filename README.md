# The Wild Oasis 🏞️

Welcome to The Wild Oasis, a comprehensive web application designed to manage hotel bookings and guest information for a serene retreat. This project represents a significant milestone in my web development learning journey, showcasing a robust implementation of modern frontend technologies.

## 🚀 Features

*   **Intuitive Booking Management:** Easily create, view, update, and delete bookings.
*   **Guest Management:** Maintain a detailed record of all guests.
*   **Cabin Management:** Oversee available cabins and their details.
*   **Authentication:** Secure user login and password management.
*   **Dark Mode Support:** Enhance user experience with a customizable dark mode.
*   **Data Visualisation:** Gain insights into booking trends with interactive charts.
*   **Responsive Design:** Accessible and functional across various devices.
*   **Error Handling:** Robust error boundaries for a seamless user experience.

##  Challenges and Learnings

* Understanding and implementation of Supabase for database management.
* Managing remote state with ReactQuery.

## 📚 Usage

Once the application is running, you can interact with its features through the web interface.

### Authentication

*   **Login:** Navigate to the login page to access the application using your credentials.
*   **Sign Up:** New users can register for an account.
*   **Password Update:** Logged-in users can update their passwords.
*   **Logout:** Securely log out of your session.

### Booking Management

*   **View Bookings:** Access the main dashboard to see all upcoming and past bookings.
*   **Create Booking:** Add new bookings for guests and cabins.
*   **Edit Booking:** Modify existing booking details.
*   **Delete Booking:** Remove a booking.

### Cabin and Guest Management

*   **View Cabins:** Browse the list of available cabins.
*   **View Guests:** See a list of all registered guests.
